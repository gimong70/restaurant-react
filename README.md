# 서울특별시 양천구 음식점 현황

공공데이터포털 API를 활용한 서울특별시 양천구 음식점 정보 조회 웹 애플리케이션입니다.

## 🚀 기능

- **전체 음식점 조회**: 양천구 내 모든 음식점 정보를 한 번에 로드
- **검색 기능**: 식당 이름 또는 주소로 검색
- **페이징**: 초기 12개 표시, "더 보기" 버튼으로 12개씩 추가 로드
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 모두 지원
- **React 기반**: React 18과 Babel을 사용한 현대적인 웹 애플리케이션

## 📋 사용 방법

1. "데이터 불러오기" 버튼을 클릭하여 음식점 데이터 로드
2. 검색창에 식당 이름 또는 주소를 입력하고 "검색하기" 클릭
3. "더 보기" 버튼으로 추가 음식점 정보 확인

## 🛠️ 기술 스택

- **React 18**: UI 라이브러리
- **Babel Standalone**: JSX 변환
- **공공데이터포털 API**: 서울특별시 양천구 음식점 현황 데이터

## 📦 배포

### GitHub Pages로 배포

1. 저장소 설정 → Pages 메뉴로 이동
2. Source: "Deploy from a branch" 선택
3. Branch: "main", "/ (root)" 선택
4. Save 클릭

배포 후 몇 분 후 `https://gimong70.github.io/restaurant-react/` 에서 접속 가능합니다.

### 다른 배포 방법

자세한 배포 방법은 [DEPLOYMENT.md](./DEPLOYMENT.md) 파일을 참고하세요.
